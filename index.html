Dock Previews

Preview all your windows directly from the dock

Window Previews in Your Dock

DockDoor brings Windows-style previews to macOS. Hover over any application in your dock to see all open windows for that app. Click the window you want or use the controls to manage windows without changing focus.

Alt+Tab Window Switcher

Switch between windows with a familiar interface

Windows-style Alt+Tab Experience

Switch between open windows just like on Windows. Press Option+Tab (Alt+Tab) to bring up the window switcher that shows live previews of all open windows. Finally bring the familiar Windows experience to your Mac.

Quickly close windows without switching focus by middle-clicking on their preview. Perfect for cleaning up your workspace while staying in your current application.

Frequently Asked Questions

Get answers to common questions about DockDoor.

Is DockDoor really free? ↓

Yes, DockDoor is completely free and open-source. You can download and use it without paying anything. If you find it useful, consider supporting development by making a donation or contributing to the project on GitHub.

Does DockDoor work with all macOS applications? ↓

DockDoor works with most standard macOS applications. However, some applications using non-standard window management (like Electron apps or apps with custom UI frameworks) might have limited functionality. We're constantly improving compatibility with various applications.

Does DockDoor impact system performance? ↓

DockDoor is designed to be lightweight and efficient. It has minimal impact on system performance and memory usage. The app runs in the background and only activates when needed, preserving your Mac's resources for other tasks.

How do I customize keybindings? ↓

You can customize keybindings in DockDoor's preferences. Open the app, click on the menu bar icon, select Open Settings, and navigate to the General tab. Scroll all the way down, click to Show Advanced Settings and find the keybind section. From there, you can change the default Option+Tab combination to any key combination that works for you.

How do I contribute to DockDoor's development? ↓

You can contribute to DockDoor in several ways:

  • Submit bug reports or feature requests on GitHub
  • Contribute code through pull requests
  • Help with translations
  • Spread the word about DockDoor
  • Support development financially through Buy Me a Coffee

Visit our GitHub repository for more information on how to contribute.
