Frequently Asked Questions

What is DockDoor?
DockDoor is an application for macOS that enhances your dock by providing quick previews and management of windows directly from the dock.
Is DockDoor compatible with my macOS version?
DockDoor is designed to work with recent versions of macOS. Please check the system requirements on our GitHub page for specific compatibility information.
How do I install DockDoor?
You can download the latest version of DockDoor from our GitHub releases page.

Alternatively, if you use Homebrew, you can install DockDoor using the command: brew install --cask dockdoor

After installation, simply launch the application, and it will integrate with your dock.
How can I customize DockDoor?
DockDoor offers various customization options through its preferences menu.
You can adjust settings such as preview size, appearance, and behavior to suit your workflow.
